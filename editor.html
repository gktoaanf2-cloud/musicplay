SYSTEM GUIDE

  1. 1. 음악 및 커버 이미지 등록
    하단 입력창에 유튜브 링크와 곡 제목을 넣어 플레이리스트를 만드세요. 추가된 곡의 우측 메뉴 중 둥근 아이콘을 클릭하면 곡마다 개별 이미지를 등록할 수 있습니다.
  2. 2. 색상 및 공통 이미지 설정
    상단의 컬러 팔레트를 클릭해 사이트 전체의 메인 색상을 자유롭게 변경합니다. 중앙의 LP판을 클릭하면 재생목록 전체에 적용되는 공통 이미지가 등록됩니다.
  3. 3. 폰트(FONT) 버튼
    FONT 버튼을 클릭하면 사이트의 영문은 Bodoni Moda, 한글은 북엔드 바탕 폰트로 조화롭게 자동 변환됩니다.
  4. 4. 생성된 페이지를 인터넷으로 공유하는 법
    ① 상단의 SHARE 버튼을 누르면 완성된 결과물이 HTML 파일로 다운로드됩니다.
    ② 무료 호스팅 사이트인 Netlify Drop (클릭)에 접속하여, 다운받은 HTML 파일을 화면 중앙 원 안에 드래그 앤 드롭 하세요. 1분 안에 누구나 접속할 수 있는 웹사이트 링크가 만들어집니다.
