SYSTEM GUIDE

  1. 1. 음악 및 커버 이미지 등록
    하단 입력창에 유튜브 링크와 곡 제목을 넣어 플레이리스트를 만드세요. 추가된 곡의 우측 메뉴 중 둥근 아이콘을 클릭하면 개별 이미지를 등록할 수 있습니다.
  2. 2. 색상 및 공통 이미지 설정
    상단의 컬러 피커를 클릭해 사이트 전체의 메인 색상을 변경합니다. 중앙의 LP판을 클릭하면 공통 커버 이미지가 등록됩니다.
  3. 3. 폰트(FONT) 버튼
    FONT 버튼을 클릭하면 영문 Bodoni Moda, 한글 북엔드 바탕 폰트로 자동 변환됩니다.
  4. 4. 인터넷 공유
    상단의 SHARE 버튼을 누르면 완성된 HTML 파일이 다운로드됩니다. 무료 호스팅(Netlify Drop 등)에 업로드해 링크를 만드세요.
