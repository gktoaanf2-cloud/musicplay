SYSTEM GUIDE

  1. 1. 음악 및 커버 등록: 하단 폼에 유튜브 링크와 곡 제목을 입력하여 플레이리스트를 생성합니다. 곡명 옆 둥근 아이콘을 눌러 개별 이미지를 매칭할 수 있습니다. (10곡 이상 등록 가능)
  2. 2. 컬러 및 커버 조율: 상단의 컬러 피커를 눌러 사이트 전체의 테마 컬러를 제어합니다. 중앙 LP판을 클릭하면 공통 커버 이미지를 등록할 수 있습니다.
  3. 3. 폰트(FONT) 토글: FONT 버튼을 누르면 영문은 Bodoni Moda, 한글은 북엔드 바탕 폰트로 조화롭게 자동 변경됩니다.
  4. 4. HTML 파일 공유 방법:
    ① 상단의 SHARE 버튼을 누르면 설정된 모든 데이터가 담긴 HTML 파일이 다운로드됩니다.
    ② 다운로드 받은 HTML 파일을 Netlify Drop (클릭) 사이트 중앙에 드래그 앤 드롭하세요. 무료 웹사이트 링크가 생성됩니다.
